Javascript Functionality

  • console.log(msg); - Logs a message to the developer console.

  • document.getElementById(someId); - returns the element that has the ID attribute with the specified value.

  • document.getElementsByClassName(someClassName); - Returns a HTMLCollection containing all elements with the specified class name.

  • document.querySelector(selector); - Returns the first element that matches a specified CSS selector(s) in the document.

  • document.querySelectorAll(selector); - Returns a static NodeList containing all elements that matches a specified CSS selector(s) in the document.

  • element.innerText = text; - sets or returns the text content of the specified node, and all its descendants.

  • element.innerHTML = html; - Sets or returns the content of an element.

  • let attrValue = element.getAttribute(attrName); - Sets local variable attrValue equal to the specified attribute value of the element 'attrName'.

  • element.setAttribute(attrName, attrValue); - Add the attrName attribute to the element, with the value of attrValue.

  • let controlValue = controlElement.value - Sets the local variable controlValue equal to the value of controlElement.
