CSS Properties

  • float
    Defines if a div or img element should float to the left or right.
    float: left | right;

  • width
    Defines the width of a div element.
    width: auto | 200px;

  • height
    Defines the height of a div element.
    height: auto | 100px;

  • margin
    Defines the margin of the four sides of an element. In order, each value correlates to: top, right, bottom, and left margins.
    margin: 50px 30px 20px 40px | auto;

  • padding
    Much like margin, defines the padding of the four sides of an element. Each value also correlates to top, right, bottom, and left paddings, respectively.
    padding: 50px 30px 20px 40px | auto;

  • display
    Allows for an element to be displayed as an inline element or as a block element.
    display: inline | block;

  • background-color
    Defines the background color of a div or body element.
    background-color: black | #000000 | rgb(0, 0 ,0);

  • background-image
    Allows the use of an image as a background image of a div or body element.
    background-image: url('URL') | radial-gradient(red, green, blue);

  • font-size
    Sets the size of the displayed font.
    font-size: small | large | 50px;

  • text-align
    Aligns content and text within an element.
    text-align: center | left | right;
