HTML Elements

  • <h1> : Used to define a header. The largest size is h1, and smallest is h6.
    <h1>HTML Elements</h1>

    HTML Elements

  • <p> : Used to define a paragraph of text. Works best for large groups of text.
    <p>This is a paragraph made up of words. This is a paragraph made up of words. This is a paragraph made up of words.</p>

    This is a paragraph made up of words. This is a paragraph made up of words. This is a paragraph made up of words.

  • <br/> : Used to make a "break" in text or add a new line. Self closed element.
    <p>These are some words.<br/> This is some more words.</p>

    These are some words.
    This is some more words.

  • <head> : Defines the head of an html webpage. Usually contains things such as title, charset, and style.
    <head>
    ..<meta charset="utf-8">
    ..<title>HTML</title>
    ..<style>
    ..</style>
    </head>

  • <body> : Defines the body of an html webpage. Usually contains the main content of what appears on a webpage.
    <body>
    ..<h3>A header in the body of a webpage</h3>
    ..<p>A paragraph in the body of a webpage.<p>
    </body>

  • <div> : Used to define a section of a webpage, allowing it to have its own styles and attributes.
    <div>This is a div in html. Giving it a class would allow it to be given CSS properties from the style section.</div>
    This is a div in html. Giving it a class would allow it to be given CSS properties from the style section.

  • <a> : Used to make links on a webpage.
    <a href="html.html">This is a link to this page</a>
    This is a link to this page

  • <img> : Used to insert images into a webpage.
    <img src="http://placekitten.com/100/100"/>

  • <title> : Used to add a title to the tab of the webpage. Appears on the tab of the page.
    <title>HTML</title>
    This webpage's title is "HTML", as shown on the tab of it.

  • <style> : Used to organize styling for a webpage.
    <style>
    body {
    text-align: center;
    background-color: white;
    }
    body {
    color: white;
    background-color: black;
    }
    </style>
