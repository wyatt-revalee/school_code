</head>

Links - when clicked, takes the user to a different webpage. Can also be used to link pictures to a webpage.
<a href="./file.txt">Name of link</a>
<a href="www.website.com">Name of link</a>
Link to this page

Image link of a cat.

<img src="cat.jpg" width="150" height="100">

Lists
Unordered (bulleted)
<ul>
----<li>A List item<li>
</ul>

Ordered Lists
<ol type='i'>
----<li>List with lowercase Roman numerals</li>
</ol>
<ol type='I'>
----<li>List with uppercase Roman numerals</li>
</ol>

Lists can also be nested in each other
<ol type='a'>
----<li>lowercase alpha</li>
----<ol type='A'>
--------<li>uppercase alpha</li>
----</ol>
</ol>

Buttons - Can be used to do things, such as execute JS functions, when clicked.
<button type="button" onclick="black()">Black</button>
<script>
function black() {
----document.body.style.backgroundColor= "black";
----document.getElementById("mainHeader").style.color = "white";
}
</script>
The above code will create a button labeled "Black" that, when clicked, will execute the function black(), which changes the background color to black, and the header with ID of "mainHeader" to white.
